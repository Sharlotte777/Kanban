# Kanban
 Установка
Скачайте app-debug.apk из репозитория.

Установите на Android 4.2+ (разрешите «неизвестные источники»).

Главный экран
Три колонки: To Do, In Progress, Done.

Поиск – сверху (фильтрация по названию, описанию, тегам).

Кнопки: Доски, Пользователи, Настройки, Обновить, + (новая задача).

Тикеты (задачи)
Действие	Жест
Создать	Нажать +, заполнить поля, Save
Редактировать	Одинарный клик по карточке
Переместить	Долгое нажатие → перетащить в другую колонку
Дублировать	Двойной клик
Удалить	Тройной клик → подтвердить
Доски / Пользователи
Создать – кнопка + на соответствующем экране.

Редактировать – одинарный клик.

Дублировать – двойной клик.

Удалить – тройной клик.

Порядок досок – перетаскивание (долгое нажатие).

Telegram уведомления
Создайте бота через @BotFather, скопируйте токен.

В приложении: Настройки → укажите User ID (логин) и Chat ID (число из https://api.telegram.org/bot<токен>/getUpdates).

В тикете укажите тот же User ID как исполнителя – придут уведомления.

Синхронизация
Все изменения тикетов автоматически синхронизируются через Firebase.

Свайп вниз или кнопка Обновить – ручная синхронизация.

Частые проблемы
Уведомления не приходят – проверьте токен и chat‑id, напишите боту первыми.

Drag‑and‑drop не работает – убедитесь, что долго удерживаете карточку.

Приложение вылетает – проверьте наличие google-services.json в папке app.
